# HIP-59 Reduce XOR filter fees - Author(s): @Frenchman , @cyborg - Start Date: 2022-04-19 - Category: Technical - Original HIP PR: https://github.com/helium/blockchain-core/pull/1303 - Tracking Issue: https://github.com/helium/HIP/issues/391 - Status: In Discussion # Summary Helium community members that host a Router instance use on-chain XOR filters to process traffic from known devices in order to significantly improve performance. This is an integrated component of a Router instance, and allows Routers and Packet Purchasers on the network to avoid spending resources on join packets from unknown devices. This HIP proposes a change for how fees are calculated when doing a XOR filter update with the goal of reducing these fees and enabling more updates to XOR filters. # Rendered View https://github.com/helium/HIP/blob/main/0059-reduce-xor-filter-fees.md
